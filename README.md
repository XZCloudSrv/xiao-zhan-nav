## ✨ 项目简介

**小战导航** 是一款由 **小战云 By 小战** 打造的、追求极致**简洁**与**美观**的个人网址导航页解决方案。

本项目基于最新的前端主流技术栈 **Vue 3.5** 和 **Nuxt 4.2** 驱动，旨在提供一个加载迅速、界面优雅、高度可定制的个人导航页面。我们特别注重用户体验和视觉效果，通过引入 **Google Fonts** 等设计元素，让您的导航页在实用之余，更具赏心悦目的美感。

> **项目状态:** 本项目目前仍在积极完善中，欢迎各位开发者和用户提出宝贵的意见和建议，共同打造更优秀的导航体验。

## 🌟 核心特性

| 特性 | 描述 |
| :--- | :--- |
| **极简主义设计** | 界面整体简洁、优雅，去除冗余元素，专注于核心导航功能。 |
| **美观字体加持** | 引入 Google Fonts，提供出色的排版和视觉体验，提升页面质感。 |
| **高性能体验** | 依托 Nuxt 4 的强大能力，实现快速加载和优秀的首屏渲染性能。 |
| **现代技术栈** | 采用 Vue 3.5、Pinia 3.0、Sass 1.9 等前沿技术，确保项目的高效开发和维护。 |
| **易于定制** | 模块化设计和 Pinia 状态管理，使得个性化配置和二次开发变得简单。 |

## 🚀 快速开始

### 1. 环境准备

请确保您的开发环境中已安装 **Node.js** (推荐 v18+) 和 **pnpm** (或 npm/yarn)。

### 2. 克隆仓库

```bash
# 从 GitHub 克隆项目到本地
git clone https://github.com/XZCloudSrv/xiao-zhan-nav.git
cd xiao-zhan-nav
```

### 3. 安装依赖

```bash
# 推荐使用 npm
# npm install

# 或者使用 pnpm
pnpm install
```

### 4. 运行开发服务器

```bash
# 启动本地开发服务器，通常在 http://localhost:3000 访问
npm run dev
```

### 5. 构建生产版本

```bash
# 构建项目用于生产环境部署
npm run build

# 预览构建结果
npm run preview
```
